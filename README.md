# Projeto Teste de Hipótese:
Este projeto consiste em uma análise exploratória de dados de um serviço de streaming de música para comparar o comportamento de consumo entre usuários de duas cidades distintas: Springfield e Shelbyville. O trabalho abrange todo o ciclo de vida da análise de dados, incluindo a limpeza de cabeçalhos, tratamento de valores ausentes (NaN), remoção de duplicados explícitos e correção de erros ortográficos em categorias de gêneros musicais (duplicados implícitos). Através de funções personalizadas e agrupamentos de dados, o projeto valida hipóteses sobre como o dia da semana e a localização geográfica influenciam a atividade dos ouvintes.

# Resultados:
Concui-se que há diferenças de uso do aplicativo entre as duas cidades com usuários em Springfield escutando mais músicas pelas sextas e menos pelas quartas, enquanto usuários em Shelbyville escutam mais pelas quartas e menos pelas segundas, além de Springfield ter mais usuários ativos do que Shellbiville. 

# Tecnologias utilizadas:
Python.

Pandas: Manipulação e tratamento de dados.

Jupyter Notebook: Ambiente de desenvolvimento.
